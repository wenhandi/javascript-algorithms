# Factorial

In mathematics, the factorial of a non-negative integer `n`, 
denoted by `n!`, is the product of all positive integers less 
than or equal to `n`. For example:
非负整数n的阶乘表示为 n!，是小于等于n的所有正整数的乘积
```
5! = 5 * 4 * 3 * 2 * 1 = 120
```
